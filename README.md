# 🧠 AI/ML Engineer Research Journey

A comprehensive, hands-on learning path to becoming an **AI/ML Engineer**. This repository documents my research, experiments, notes, and projects as I work towards mastering the full AI/ML engineering stack.

> **Mission**: Move from theory to production - building real ML systems, not just running Jupyter notebooks.

---

## 🎯 Learning Objectives

- Master **MLOps** pipelines and deployment strategies
- Build production-ready ML systems (APIs, services, observability)
- Understand **LLM engineering** practically (RAG, agents, fine-tuning)
- Master **experimental design** for ML research (metrics, ablations, benchmarks)
- Deploy models to various platforms (cloud, edge, mobile)

---

## 📚 Quick Start

1. **Clone this repo**
   ```bash
   git clone https://github.com/jarrettj/ai-ml-engineer-research.git
   cd ai-ml-engineer-research
   ```

2. **Start with the [Research Path](docs/RESEARCH-PATH.md)**
   - Read the learning curriculum
   - Follow the milestone checkboxes
   - Track your progress in **Exercises** and **Projects**

3. **Join the community**
   - Open issues for questions
   - Create PRs with completed exercises
   - Share your projects and results!
